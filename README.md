# Hakai-KEYGEN
a simple C++ console app that makes keys for Hakai SDK and other Hakai related tools.

### Please note
These do not generate LIVE keys for any Hakai tool.
they are pretty much useless unless u use them for your own AUTH setup. 
---
## Requirements.
[Microsoft Visual C++ Redistributable](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

Probably advised you get the latest if u dont it at all.

# How to build (for noobs)

Download the source 

Download Visual Studio    [HERE](https://visualstudio.microsoft.com/downloads/)

you WILL need the packages for C++ development.

after setting up add the project to yours.

and just build

thats literally it
